# 智能客服系统使用指南

## 功能概述

意念科技智能客服系统是一个基于 AI 的自动客服解决方案，为用户提供 24/7 的在线咨询服务。

### 核心功能

1. **智能对话**：基于 AI 的自然语言处理，理解用户意图并给出专业回答
2. **快速回复**：针对常见问题提供秒级快速回复
3. **FAQ 知识库**：结构化的常见问题分类浏览
4. **多入口访问**：支持页面导航、悬浮按钮等多种访问方式
5. **对话历史**：保存对话历史，支持上下文理解

## 用户使用指南

### 访问方式

#### 方式一：通过导航菜单

1. 登录平台后，在顶部导航栏点击"智能客服"
2. 进入完整的客服页面，享受全功能服务

#### 方式二：通过悬浮按钮

1. 在任意页面右下角找到客服悬浮按钮（绿色/蓝色圆形按钮）
2. 点击按钮，可以选择：
   - 快速对话：打开迷你聊天窗口，快速提问
   - 完整客服：跳转到完整客服页面

### 对话功能

#### 发送消息

- **输入框**：在底部输入框输入您的问题
- **快捷按钮**：点击常用问题标签快速提问
- **回车发送**：按 Enter 键快速发送消息（Shift+Enter 换行）

#### 快速回复

系统内置了常见问题的快速回复，包括：

| 关键词 | 自动回复内容 |
|--------|-------------|
| 价格 | 订阅计划价格信息 |
| 订阅 | 订阅计划和升级说明 |
| 免费 | Free 计划功能说明 |
| 实时 | 实时行情说明 |
| API | API 使用方法和限制 |
| 账单 | 账单查询说明 |
| 密码 | 修改和重置密码 |
| 预警 | 预警功能说明 |
| 自选 | 自选股管理说明 |
| AI | AI 分析功能说明 |
| 联系 | 人工客服联系方式 |
| 风险 | 投资风险提示 |
| 企业 | Enterprise 计划介绍 |

#### 智能对话

对于更复杂的问题，客服会调用 AI 模型进行智能回复，支持：

- 自然语言理解
- 多轮对话
- 上下文记忆
- 个性化建议

### FAQ 知识库

1. 在左侧边栏选择问题分类：

### 对话管理

#### 清空对话

1. 点击页面右上角的"清空对话"按钮
2. 确认清空后，所有对话历史将被删除

#### 查看历史

- 对话历史在当前会话中自动保存
- 刷新页面后历史会清空（如有需要可扩展为持久化存储）

## 开发者指南

### API 端点

#### POST /api/customer-service/chat

发送客服消息

**请求参数：**
```json
{
  "message": "用户消息内容",
  "history": [
    {
      "role": "user",
      "content": "上一条用户消息"
    },
    {
      "role": "assistant",
      "content": "上一条客服回复"
    }
  ]
}
```

**响应示例：**
```json
{
  "success": true,
  "reply": "客服回复内容",
  "tokensUsed": 150,
  "model": "gpt-4"
}
```

#### GET /api/customer-service/faq

获取所有常见问题

**响应示例：**
```json
{
  "success": true,
  "data": [
    {
      "category": "订阅相关",
      "questions": [
        {
          "question": "问题标题",
          "answer": "问题答案"
        }
      ]
    }
  ]
}
```

#### GET /api/customer-service/faq/:category

根据分类获取常见问题

**路径参数：**
- `category`: 分类名称

#### GET /api/customer-service/categories

获取所有分类名称

**响应示例：**
```json
{
  "success": true,
  "data": ["订阅相关", "功能使用", "API和账单"]
}
```

### 组件使用

#### CustomerServiceButton 悬浮按钮

```vue
<template>
  <CustomerServiceButton
    :button-size="'large'"
    :has-unread="false"
  />
</template>

<script setup>
import CustomerServiceButton from './components/CustomerServiceButton.vue'
</script>
```

**Props：**
- `buttonSize`: 按钮大小（`'large'` | `'default'` | `'small'`，默认：`'large'`）
- `hasUnread`: 是否有未读消息（默认：`false`）

**暴露方法：**
- `open()`: 打开客服
- `close()`: 关闭客服

### 自定义配置

#### 修改快速回复关键词

编辑 `server/services/customerServiceService.js` 中的 `getQuickReply` 方法：

```javascript
const quickReplies = {
  '你的关键词': '回复内容',
  // 添加更多...
};
```

#### 修改 FAQ 内容

编辑 `server/services/customerServiceService.js` 中的 `getFAQ` 方法，修改或添加常见问题。

#### 修改系统提示词

编辑 `buildSystemPrompt` 方法，调整客服的回答风格和知识范围。

## 技术架构

### 后端

- **Express.js**: Web 框架
- **CustomerServiceService**: 客服业务逻辑层
- **AI 服务集成**: 调用 AI 模型进行智能对话
- **JWT 认证**: 用户身份验证

### 前端

- **Vue 3**: 前端框架
- **Element Plus**: UI 组件库
- **Vue Router**: 路由管理
- **Axios**: HTTP 请求

### 核心流程

1. 用户发送消息
2. 检查快速回复关键词
3. 如果匹配关键词，返回快速回复
4. 如果不匹配，调用 AI 服务
5. AI 服务根据对话历史和系统提示词生成回复
6. 返回回复给用户

## 最佳实践

### 提问建议

1. **明确具体**：描述清楚您的问题
2. **提供上下文**：如果是技术问题，说明具体情况
3. **先查 FAQ**：在提问前先查看常见问题，可能已有答案
4. **使用关键词**：快速回复的关键词可以快速获取标准答案

### 客服优化建议

1. **定期更新 FAQ**：根据用户反馈更新常见问题
2. **优化关键词**：根据实际对话调整快速回复关键词
3. **训练 AI 模型**：优化系统提示词，提升回答质量
4. **监控对话**：分析对话数据，发现用户痛点

## 注意事项

1. **隐私保护**：客服对话不会记录敏感信息（如密码、Token）
2. **Token 计费**：智能客服调用 AI 模型可能会消耗 Token（取决于配置）
3. **功能限制**：某些复杂问题可能需要联系人工客服
4. **投资建议**：客服不提供投资建议，所有信息仅供参考
